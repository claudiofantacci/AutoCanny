AutoCanny
===========

The AutoCanny library provides out-of-the-box, parameter free, [Canny edge detector](https://en.wikipedia.org/wiki/Canny_edge_detector).

AutoCanny works on Linux, macOS and Windows.

## Dependencies
AutoCanny depends on [OpenCV](https://github.com/opencv/opencv) (>= 3.2.0).

## Build the library
### Linux / macOS
~~~shell
git clone https://github.com/claudiofantacci/AutoCanny
cd AutoCanny
mkdir build && cd build
cmake ..
make
[sudo] make install
~~~

## Use the library
Once the library is installed, add to your project `CMakeLists.txt`
~~~cmake
...
find_package(AutoCanny REQUIRED)
...
target_link_libraries(<target> AutoCanny)
...
~~~
